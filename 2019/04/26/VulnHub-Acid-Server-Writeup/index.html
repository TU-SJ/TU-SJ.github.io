总结

存在的问题

1.在web漏洞利用中耗时太多,然而漏洞利用并不难
2.获取shell后不知道怎么提权,随意翻看文件且毫无头绪.

解决

基于目标进行渗透：

  • 利用web漏洞的目的是拿到shell,在此基础上以最快、最简洁、最隐蔽的方式利用漏洞获取shell
